# 集测报告约束

## 格式
```markdown
# 集测报告: [任务编号 - 任务名称]

## 测试执行概要

- 测试日期: [日期]
- 测试环境: [环境信息]
- Python 版本: [版本]
- pytest 版本: [版本]
- 测试结果: **全部通过** / **不通过（失败数不为0）**

## 测试统计

- 测试文件数: N
- 测试类数: N
- 测试用例数: N
- 通过数: N
- 失败数: 0
- 跳过数: 0
- 总执行时间: Xs
- 通过率: X%

## 测试执行详情

### [测试文件名]

```
[pytest输出]
```

**结果: N/N 通过**
```
