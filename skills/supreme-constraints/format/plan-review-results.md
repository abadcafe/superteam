# 规划审查报告约束

## 格式
```markdown
# 规划审查报告: [与规划文档标题保持一致]

审查时间: [时间]

## 审查结果

### 🔴 严重问题（必须修复）

#### 问题 N: [问题标题]
**位置**: [规划文档章节]
**描述**: [问题描述]
**设计依据**: [引用架构文档相关内容（如有，否则填无）]
**备注**: [说明为何属于严重问题]

### 🟡 普通问题（必须修复）

#### 问题 M: [问题标题]
**位置**: [规划文档章节]
**描述**: [问题描述]
**设计依据**: [引用架构文档相关内容（如有，否则填无）]
**备注**: [说明为何属于普通问题]

## 总结
- 严重问题: N 个
- 普通问题: M 个

## 结论
- [ ] 有问题，需修复
- [ ] 无问题
```
